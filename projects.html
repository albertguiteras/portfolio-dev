LAB Artistas

Proyecto de desarrollo web multipagina construido para consolidar fundamentos de maquetación. Desarrollado puramente con HTML5 y CSS3, este proyecto destaca por la experimentación con diseño visual, uso de display: flex para estructurar listas complejas, anidamiento de elementos, y presentación de datos estadísticos o discografías mediante tablas personalizadas. LAB Robo AI

Desarrollo de una Single Page promocional enfocada en la presentación de un producto tecnológico. Este proyecto destaca por el uso avanzado de CSS y Flexbox para construir menús de navegación horizontales y estructurar el layout mediante el control preciso de las propiedades display (block, inline-block y flex). Incluye integración tipográfica con Google Fonts y el diseño de botones Call to Action (CTA) interactivos con efectos :hover, sombras y bordes estilizados.

LAB Dragon Ball

Proyecto Front-end enfocado en el dominio avanzado de CSS3 y Flexbox. Consiste en un sistema de tarjetas responsive que utiliza propiedades de flujo flexible (flex-wrap, flex-direction) para adaptarse a cualquier pantalla. El proyecto destaca por la implementación de microinteracciones UI mediante pseudo-clases (:hover), aplicando transformaciones 2D complejas (escalado, rotación, traslación) y transiciones fluidas para mejorar la experiencia de usuario.

LAB Sobre mí

Proyecto de presentación web ("About Me") desarrollado para dominar los fundamentos del diseño fluido. La interfaz utiliza CSS Flexbox de forma intensiva para estructurar cabeceras, galerías de imágenes y listados de intereses, aplicando propiedades avanzadas como justify-content, align-items y flex-wrap para lograr un layout adaptable. El proyecto destaca por incluir el flujo de despliegue a producción, estando alojado y accesible en vivo a través de GitHub Pages.
